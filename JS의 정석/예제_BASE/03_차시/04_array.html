Array(배열)

배열은 단순히 숫자나 문자를 변수명 하나에 각각 담는 것을 넘어 비슷한 종류의 값들을 하나의 변수명 안에 저장할때 객체를 만들수도 있음 
        배열은 비슷한 성격의 여러 데이터들을 한 저장소에 저장해 놓고 
        필요할 때 데이터를 꺼내서 사용할 때 생성합니다.   

        새배열 만드는 법
        1.var 이름 = new Array();
        2.var 이름 = [];

        배열명[0] = 값;
        배열명[인덱스번호]=값;

        값 확인하기
        배열명[1]

        배열의 개수 
        배열명.length

join(합치다)

        배열의 값을 하나의 문자열로 합쳐준다!
        (join);
        사용법-
        배열명.join('구분자');

reverse(반대로)

        원소의 정렬를 역순 변경(원본 수정)
        배열명.reverse();

sort(섞다)

        원소들을 오른차순, 내림순 정렬 (원본 수정)
        배열명.sort();

push(밀어넣다)

배열을 마지막에 추가해주는 메서드

pop(터트리다)

unshift(배열값들의 맨앞에 추가해서 출력)

shift(맨앞에 추가된 배열값 제거 메서드)

slice(구간을 정하고 그 구간의 값을 복사)

기존변수.slice(구간인덱스번호,인덱스번호로부터 몇번째 배열까지 복사해올지 정하는 수)

splice(기존배열값 오려내기)

concat(배열 값을 병합해줌 )

        원본.concat(새로운배열)

filter

        조건에 부합하는 원소들을 추출하여 새로운 배열을 생성
