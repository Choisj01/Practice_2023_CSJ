변수

값을 담는 그릇! 
        변수 선언 종류(3가지)
        var 변수명 = 값; 선언하면서 값을 초기화(할당), 어디에서든 사용가능, 함수내에서 지역변수, 호이스팅(끌어올려짐)
        let 변수명 = 값; 블록범위 지역변수   {}-블록범위
        값을 지정후 변경가능
        -> 위의 두개의 선언 변수는 값을 지정 후 변경 가능

        const 변수명 = 값; const 상수, 값 변경 불가

변수 타입

        기본타입 : 숫자, 문자, 불리언(true/false), null(빈값) , undefined 등의 원시데이터로 단순화 할 수 없는 값
        참조타입: 복합적인 값 (예- 배열, 함수 , 정규 표현식들)
