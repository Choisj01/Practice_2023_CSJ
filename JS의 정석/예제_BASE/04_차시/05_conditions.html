조건문

조건문은 굉장히 다양하게 활용가능함. 자바스크립트로 웹페이지의 요소를 선택하거나 효과를 적용할때 , 그 해당 이벤트에 기준을 넘어가는지 안넘어가는지 조건을 설정해 참과 참이 아닐때 다른 효과를 구현하기 위해 사용함! 
        if(조건){ 
            참일 때 할일;            
           } else {            
            거짓일 때 할일;            
        }

switch

    조건의 매칭되는 케이스가 있다면 해당 블록의 코드를 실행.(해당 케이스에 맞는 코드 실행)

    switch(조건){
        case a:
         조건의 결과가 a일때 할일;
         break;
        case b: 
        조건의 결과가 b일때 할일;
        break;
    }

    //a 의 조건에 부합하면 결과가 실행되고 그 이후에는 조건 해당없으면 브레이크!
    그런뒤에 다시 b조건에 부합하면 결과 실행되고 이후에 b조건에 해당없으면 다시 브레이크!
    조건의 값에 따라 결과를 다양하게 구현할 수 있음
